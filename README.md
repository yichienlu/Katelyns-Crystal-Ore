## UX 開發流程

### User Story 使用者需求

#### 顧客
- 做脈輪測驗，了解自身脈輪能量
- 了解晶礦知識
- 自行設計礦石手串
- 編輯會員資料

- 查看訂單商品設計、製作晶礦手串
- 管理訂單狀態
- 查看客戶資料
